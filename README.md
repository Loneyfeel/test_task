# Тестовое задание — страница сайта

## Технологии
- **React** + **TypeScript**
- **Emotion** для стилизации компонентов

## Реализовано
- Несколько секций сайта
- Адаптивная верстка под разные разрешения
- **Боковая панель (sidebar)** с дропдаунами и чатом в реальном времени
- Моковая генерация дропов и сообщений чата
- Рандомизированные карточки товаров

## Особенности
- Интерактивный интерфейс благодаря React
- Дропы и чат работают с имитацией реального времени
- Современная CSS-in-JS стилизация через Emotion

## Трудности / Замечания
- Шрифт не полностью соответствует дизайну, хотя установлен корректно
- Реализация заняла больше времени, чем предполагалось при первичной оценке задания
